# カード効果一覧（ドミニオン基本第二版）

このドキュメントは、ドミニオン基本第二版に含まれる全カードの効果を記載しています。

## 財宝カード

| カード名 | コスト | 効果 |
|----------|--------|------|
| 銅貨（Copper） | 0 | +1コイン |
| 銀貨（Silver） | 3 | +2コイン |
| 金貨（Gold） | 6 | +3コイン |

## 勝利点カード

| カード名 | コスト | 勝利点 |
|----------|--------|--------|
| 屋敷（Estate） | 2 | 1点 |
| 公領（Duchy） | 5 | 3点 |
| 属州（Province） | 8 | 6点 |

## 呪いカード

| カード名 | コスト | 勝利点 |
|----------|--------|--------|
| 呪い（Curse） | 0 | -1点 |

## 王国カード（アクション）

### コスト2

| カード名 | 効果 |
|----------|------|
| 地下貯蔵庫（Cellar） | +1アクション<br>手札を好きな枚数捨て、同数ドローする |
| 礼拝堂（Chapel） | 手札から最大4枚を廃棄する |
| 堀（Moat） | +2カード<br>（リアクション：アタックカードを無効化） |

### コスト3

| カード名 | 効果 |
|----------|------|
| 先触れ（Harbinger） | +1カード、+1アクション<br>捨て札からカードを1枚選び、デッキの上に置く |
| 商人（Merchant） | +1カード、+1アクション<br>このターンに初めて銀貨をプレイした時、+1コイン |
| 家臣（Vassal） | +2コイン<br>デッキからカード1枚を公開し、捨て札に置く。それがアクションカードなら、プレイしてもよい |
| 村（Village） | +1カード、+2アクション |
| 工房（Workshop） | コスト4以下のカードを1枚獲得する |

### コスト4

| カード名 | 効果 |
|----------|------|
| 官僚（Bureaucrat） | 銀貨1枚を獲得し、デッキの上に置く<br>アタック：他のプレイヤーは手札から勝利点カード1枚を公開し、デッキの上に置く |
| 庭園（Gardens） | 0点<br>（ゲーム終了時、カード10枚につき1点） |
| 民兵（Militia） | +2コイン<br>アタック：他のプレイヤーは手札が3枚になるまで捨てる |
| 金貸し（Moneylender） | 手札から銅貨1枚を廃棄してもよい。廃棄した場合、+3コイン |
| 密猟者（Poacher） | +1カード、+1アクション、+1コイン<br>空のサプライの山札1つにつき、手札からカード1枚を捨てる |
| 改築（Remodel） | 手札からカード1枚を廃棄する。廃棄したカードのコスト+2以下のカードを1枚獲得する |
| 鍛冶屋（Smithy） | +3カード |
| 玉座の間（Throne Room） | 手札からアクションカード1枚を選び、2回プレイする |

### コスト5

| カード名 | 効果 |
|----------|------|
| 山賊（Bandit） | 金貨1枚を獲得する<br>アタック：他のプレイヤーはデッキから2枚公開し、銅貨以外の財宝カード1枚を廃棄する。残りは捨て札に置く |
| 議事堂（Council Room） | +4カード、+1購入<br>他のプレイヤー全員は+1カード |
| 祝祭（Festival） | +2アクション、+1購入、+2コイン |
| 研究所（Laboratory） | +2カード、+1アクション |
| 書庫（Library） | 手札が7枚になるまでドローする |
| 市場（Market） | +1カード、+1アクション、+1購入、+1コイン |
| 鉱山（Mine） | 手札から財宝カード1枚を廃棄する。廃棄したカードのコスト+3以下の財宝カードを1枚獲得し、手札に加える |
| 歩哨（Sentry） | +1カード、+1アクション<br>デッキから2枚を見る。それらを廃棄、捨て札、またはデッキの上に戻す（順不同） |
| 魔女（Witch） | +2カード<br>アタック：他のプレイヤーは呪い1枚を獲得する |

### コスト6

| カード名 | 効果 |
|----------|------|
| 職人（Artisan） | コスト5以下のカードを1枚獲得し、手札に加える。手札からカード1枚をデッキの上に置く |
